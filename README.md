# Music Player üéµ

A beautiful and functional local music player built with Flutter. This project was developed as part of a Flutter Internship Assignment to demonstrate audio playback capabilities, file handling, and UI design.

## ‚ú® Features

* **MP3 Selection:**
    * Scan device storage for audio files automatically.
    * Manually pick specific MP3 files using a file picker.
* **Audio Playback:**
    * Play, Pause, and Stop functionality.
    * Seek bar with real-time position updates.
    * Next/Previous track controls with Playlist support.
    * Shuffle functionality.
* **Rich UI:**
    * Extracts and displays **Album Art** from audio metadata.
    * Displays Track Title and Artist Name.
    * **Animated Waveform** visualizer.
    * Modern dark theme with cyan gradients and glassmorphism effects.
    * Custom animated Play/Pause button.
* **Permissions:** Handles Android storage permissions (supports Android 13+ granular audio permissions).

## üõ†Ô∏è Tech Stack & Libraries Used

* **Framework:** [Flutter](https://flutter.dev/)
* **Audio Playback:** [just_audio](https://pub.dev/packages/just_audio) - Handles the core audio logic.
* **File Picking:** [file_picker](https://pub.dev/packages/file_picker) - For selecting MP3 files from storage.
* **Permissions:** [permission_handler](https://pub.dev/packages/permission_handler) - To request storage/audio access.
* **Metadata:** [audio_metadata_reader](https://pub.dev/packages/audio_metadata_reader) - To extract album art, titles, and artist names.
* **Path Manipulation:** [path](https://pub.dev/packages/path) - For filename and extension handling.
